# 97 World — Design Sector (Website)

A multi-file static site. No build step — just open index.html, or host the folder anywhere.

## Files
- index.html — page structure
- css/styles.css — all styling (Helvetica)
- js/data.js — services + portfolio content (mirrors the 97 World Notion hub)
- js/script.js — intro animation, pricing calculator, WhatsApp order form

## Edit content
Change prices, services or portfolio in js/data.js. The same content lives in your Notion hub ("97 World — Design Sector HQ") so you can keep both in sync.
